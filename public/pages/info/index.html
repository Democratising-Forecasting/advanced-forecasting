Software

You first need to download and install:

  • Latest R (v4.0.4)
  • Latest RStudio

Open RStudio, and type the following command in the R console:

install.packages("tidyverse")

We’ll use a collection of R packages called tidyverse throughout the course.

We’ll use a collection of R packages called fpp3 throughout the course.
